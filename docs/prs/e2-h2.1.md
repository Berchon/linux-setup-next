## Resumo
Implementa a história E2/H2.1 (Rectangle), cobrindo as tasks T2.1.1, T2.1.2 e T2.1.3 com primitive reutilizável para preenchimento, borda e título com clipping.

## O que foi feito
- Criado `src/components/rectangle.sh` com render de preenchimento de área (`T2.1.1`) e clipping no viewport.
- Adicionado suporte a borda `none|single|double` com fallback ASCII-safe e API `rectangle_render` (`T2.1.2`).
- Implementado título opcional com clipping para largura disponível (com borda e sem borda) (`T2.1.3`).
- Criado/expandido teste de componente em `tests/component/rectangle_fill_test.sh` para validar preenchimento, borda e título.
- Atualizado `CHANGELOG.md` com entradas de rastreabilidade para E2/H2.1 com hashes dos commits.

## Validação executada
- `for t in tests/unit/*.sh tests/component/*.sh; do bash "$t"; done`

## Impacto
- Introduz a primeira primitive de componentes base (E2), habilitando composição futura de `Panel`, `Menu`, `Modal` e `Toast`.
- Não altera runtime de terminal nem pipeline de diff existente; impacto restrito à nova camada de componentes.
- Próximo passo imediato: iniciar E2/H2.2 (Shadow).
